# Nort Cyber Security — Defend the Infinite

Elite Tron Legacy / neon cybersecurity experience. Cinematic single-page portal built with Next.js, Tailwind CSS, Framer Motion, and React Three Fiber.

## Stack

- **Next.js 15** (App Router)
- **Tailwind CSS** — neon design system
- **Framer Motion** — loading sequence, scroll reveals, micro-interactions
- **React Three Fiber + Drei** — holographic 3D shield hero
- **Canvas 2D** — live command lattice / network globe

## Run

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

```bash
npm run build && npm start
```

## Sections

1. **Loading** — Grid assembly, logo construction, energy surge
2. **Hero** — Interactive 3D shield, threat vaporization, dual CTAs
3. **Solutions** — Four glassmorphic capability cards
4. **Intelligence** — Live threat lattice + live stats
5. **Architecture** — Four-layer stack of light
6. **Secure Uplink** — Terminal-style encrypted contact form
7. **Footer** — Circuit board trails + brand lockup
